
The workspace contains two folders by default, where:

- `lib`: the folder to maintain dependencies

El siguiente proyecto es un sistema de gestión de bibliotecas personales en el que se puede agregar y eliminar libros. De igual forma, los libros tienen un atributo de estado de lectura para hacer un rastreo a los hábitos de lectura de los usuarios. También hay una implementación de sistemas de recompensas para fomentar el hábito de la lectura en los usuarios por medio de la gamificación.
El proyecto está dividido en cuatro carpetas: gui, app, img y modelo. La carpeta img contiende las imágenes dentro del proyecto. app contiende la clase App.java, la cual es el main del mismo. Por otro lado, modelo contiene todo lo referente al backend y la base de las clases que se relacionan dentro de la biblioteca y la persistencia de datos. Por último, la carpeta gui contiene la integración de la interfaz gráfica con el backend.

The followed project is system of personal library management, in which user can add and delete books. Thus, books have a reading state attribute in order to track user's reading habits. In addition, there is a reward system to increase the reading habit in users with the use of gamification.
The project is divided into four folders: gui, app, img and modelo. img folder contains project's images. app had the App.java class, that is our main. Addittionally modelo folder's has all backend logic and perdurance of data. Finally, gui contains the integration between Graphical User Interface and backend.
